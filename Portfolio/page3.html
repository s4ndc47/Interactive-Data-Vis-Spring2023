About Data Integration

In the raw data file, certain data points were initially represented as percentages. To ensure consistency, I had convert these percentages into actual numerical values. This will involve utilizing the total population count obtained from the U.S. Census data and multiplying it by the respective percentage values related to sex and age etc., thereby deriving the actual numbers.

Additionally, since each file corresponds to a specific year, it is necessary to integrate these files into a new table encompassing data from 2011 to 2021. This consolidation will enable a comprehensive analysis of the longitudinal trends within the dataset.

Data Source: U.S. Census Bureau, (2011-2021 except 2010).
